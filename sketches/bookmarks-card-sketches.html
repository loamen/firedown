Recommendation: A (counter subtitle). It is literally the same shape as Downloads / Vault, so the home reads as a uniform stack of destinations — exactly the KISS rule you flagged. The trailing-favicon variant (C) is the strongest "more informative" option and still keeps the row shape, but it introduces a new visual element nothing else in the stack has, and favicon fetch/cache adds load-order complexity. B trades a stable subtitle format for a noisier one. D breaks subtitle parity with the siblings.

One small refinement for A: hide the subtitle when count = 0 (title-only "Bookmarks"), matching how Safe Folder hides "0 items saved" — so a fresh install reads cleanly without a misleading "0 bookmarks" line.
